# 03. 슬래시 커맨드 완전 정리

Claude 입력창에서 `/`로 시작하는 명령어를 쓰면, 대화만 하는 것보다 훨씬 빠르게 작업할 수 있어요.
형은 처음에 이것만 익혀도 충분해요.

---

## 제일 먼저 기억할 것

입력창에서 `/`를 치고 `Tab`을 누르면 자동완성이 떠요.
이름이 정확히 기억 안 나도 괜찮아요.

```text
/
```

여기서 조금 입력하고 `Tab` 눌러보면 돼요.

---

## 기본 커맨드

| 커맨드 | 언제 써요? | 한 줄 설명 |
|--------|------------|-------------|
| `/help` | 뭐가 있는지 모르겠을 때 | 도움말 보기 |
| `/clear` | 대화가 꼬였을 때 | 현재 대화를 비우고 다시 시작 |
| `/compact` | 대화가 너무 길어졌을 때 | 긴 맥락을 짧게 압축 |
| `/model` | 다른 모델로 바꾸고 싶을 때 | 현재 모델 확인/변경 |
| `/exit` | 세션을 마칠 때 | Claude 종료 |

---

## 유용한 커맨드

| 커맨드 | 언제 써요? | 예시 |
|--------|------------|------|
| `/plan` | 큰 작업 전에 순서를 정리할 때 | `/plan 숫자 맞히기 게임을 단계로 쪼개줘` |
| `/skill` | 스킬 목록이 궁금할 때 | `/skill 지금 설치된 스킬 보여줘` |
| `/learner` | 반복 작업을 스킬로 뽑고 싶을 때 | `/learner 방금 작업 패턴을 스킬로 정리해줘` |
| `/setup-local` | 내 취향을 개인 설정으로 저장할 때 | `/setup-local 답변은 짧고 한국어로 해줘` |

---

## 커스텀 커맨드는 어떻게 만들어요?

개요만 알면 돼요.
커스텀 스킬은 보통 이런 위치에 들어가요.

```text
.claude/skills/이름/SKILL.md
```

즉, 폴더 하나 만들고 그 안에 `SKILL.md`를 넣으면 돼요.
이 파일에 "이 스킬은 언제, 어떻게 동작해라"를 적는 구조예요.

> `SKILL.md`는 쉽게 말하면 AI용 사용 설명서예요.

## 이 레포에서 먼저 써볼 것

이 레포에는 `/onboard-donghyun`이 있어요.
형이 완전 초보 기준으로 한 단계씩 따라가게 만드는 온보딩 스킬이에요.

처음엔 이걸 먼저 써보면 좋아요.

```text
/onboard-donghyun
```

이 스킬은 한 번에 많은 걸 시키지 않고, **지금 단계에서 할 일 1개만** 보여주는 쪽으로 맞춰져 있어요.
