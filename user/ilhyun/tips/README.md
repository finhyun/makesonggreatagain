# tips — 실전 팁 모음

## 뭐가 들어 있나요?

| 파일 | 한 줄 소개 |

## 추천 읽기 순서

1. `04-vibe-coding.md`
2. `01-conversation.md`
3. `03-slash-commands.md`
4. `02-memory-system.md`

이 순서를 추천하는 이유는 간단해요.
먼저 마음가짐을 잡고, 그다음 대화법을 익히고, 그다음 도구를 쓰는 법을 보고, 마지막에 기억 시스템까지 붙이면 덜 헷갈려요.
