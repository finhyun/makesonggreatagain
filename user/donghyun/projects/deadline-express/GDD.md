# 데드라인 익스프레스 (Deadline Express)
**부제:** 흐름(Flow)을 지배하는 자가 황무지를 돌파한다
---
## 1. 기획 배경 및 의도
본 프로젝트는 **"왜 디펜스 게임은 항상 적이 올 때까지 기다려야 하는가?"** 라는 질문에서 시작되었습니다.
기존 디펜스 장르의 정형화된 문법인 **'웨이브(Wave)'** 시스템을 과감히 제거하고, '연속적인 흐름'을 핵심 축으로 삼습니다.
- **웨이브가 없는 이유:** 적군이 무리지어 '오는 것'이 아니라, 플레이어가 적진을 향해 '달려가는' 구조를 만들기 위함
- **기획 의도:** 게임이 정해준 타이밍에 싸우는 것이 아니라, 플레이어가 기차의 속도를 조절함으로써 스스로 교전의 밀도와 난이도를 결정하게 만드는 **'자율적 템포 디펜스'** 지향
- **MVP 검증 관점:** 이 거대한 비전은 먼저 **5분 1노드 캡슐**로 분리되어 검증되었습니다.
- **v2의 역할:** v1의 감성적 비전은 보존하되, 실제 MVP에서 확인된 수치·구조·흥미축을 공식 설계 기준으로 편입합니다.
---
## 2. 개요 (Vision)
| 항목 | 내용 |
|------|------|
| 장르 | 2D 사이드뷰 이동형 디펜스 (Continuous Flow Defense) |
| 시점 | 2D Side-View (기차 단면을 옆에서 보는 방식) |
| 핵심 컨셉 | 멈추지 않는 황무지 질주 속, 엔진과 화력을 동시에 관리하는 요새 운영 |
| MVP 승리 조건 | 5분 타이머 0 도달 또는 진행도 100% 도달, 기관차 생존 |
| MVP 패배 조건 | 기관차 HP 0 또는 연료 0 |
| 장기 승리 조건 | 기관차가 파괴되지 않고 최종 목적지에 도착 |
**로그라인:**
> "설국열차의 생존과 매드맥스의 질주가 만난 '무박자' 디펜스 — 1대의 기관차와 3종의 모듈로 구성된 요새를 이끌고, 빨리 달리면 엔진이 터지고 천천히 가면 목숨이 위태로운 극한의 딜레마를 뚫고 나가라."
MVP는 이 로그라인을 60분 풀런이 아니라 **5분 한 노드 클리어 미니런**으로 압축합니다.
따라서 v2에서의 기준은 두 층으로 나뉩니다.
| 층위 | 의미 |
|------|------|
| MVP | 5분 동안 한 노드를 돌파하며 코어 DNA를 검증하는 플레이어블 캡슐 |
| Post-MVP | 다중 노드, 정비소, 메타 진행으로 확장되는 풀런 비전 |

## 3. 절대 규칙 (The Grand Law)
> **"빨리 달리면 적을 따돌릴 수 있지만 엔진이 타버리고, 천천히 가면 자원을 모을 수 있지만 적에게 포위당한다."**
이 게임의 모든 전략적 판단은 아래 물리 법칙을 따릅니다.
| 상태 | 장점 | 단점 |
|------|------|------|
| 고속 주행 | 적 사격 범위 조기 이탈, 돌파감, 위기 탈출 | 연료 급소모, 엔진 과열, 채집 흡입 중단 |
| 저속 주행 | 자원 자동 흡입 안정화, 냉각 여유, 정비 판단 가능 | 적에게 오래 노출, 포위·침입 위험 증가 |
| 정지 | 흡입 반경 최대, 열 증가 없음 | 연료는 최소 소모되고 이동 진척이 멈춤 |
MVP 기준 속도·열·연료 삼각구도는 다음 수치를 따른다.
| 속도 | 코드값 | 연료 소모 (초당) | 열 증가 (초당) | 채집 흡입 반경 |
|------|--------|------------------|----------------|----------------|
| 정지 | 0 | 0.5 | 0 | 220 px |
| 저속 | 1 | 1.0 | 0.8 | 160 px |
| 중속 | 2 | 2.0 | 3.2 | 100 px |
| 고속 | 3 | 4.0 | 7.2 | 0 px |
- 열 증가 공식: `speed^2 * 0.8` per sec
- 기본 열 감소: `1.2` per sec
- 유지보수칸 냉각 버프: `1.5x`
- 냉각수 사용: 열 `-40%`
- 열 100% 도달: 무작위 모듈 화재 발생
- 화재 지속: 15초 DoT
- 속도는 전투 중에도 실시간 조절 가능
- 고속은 생존을 앞당기지만 채집을 포기시키고, 저속은 자원을 부르지만 전장을 길게 만든다.
이 절대 규칙은 v1의 철학이자, MVP에서 실제 입력 긴장감을 만든 가장 중요한 엔진입니다.

## 4. 기차 기본 구성: 1 + 3 시스템
### A. 머리: 기관차 (The Locomotive) — 고정
- 기차의 맨 앞, 엔진이 위치한 심장부
- 연료탱크, 열 게이지, 속도 레버 담당
- **기관차 HP 0 또는 연료 0 = 게임 오버**
- MVP HP: 200
- MVP 외벽 장갑: 100
### B. 몸통: 3가지 기본 모듈
MVP에서는 1 기관차 + 3 모듈이 고정 배치됩니다.
Post-MVP에서는 정비소에서 순서 재배치가 가능한 확장 대상으로 남겨둡니다.
| 모듈 | HP | 외벽 장갑 | MVP 역할 |
|------|----|-----------|----------|
| 전투칸 (Combat) | 120 | 80 | 외부 포탑 1문. 자동 60%, 수동 100%+크리 사격 |
| 유지보수칸 (Maintenance) | 100 | 60 | 수리 드론 1기, 냉각 1.5배 버프 |
| 채집칸 (Harvesting) | 100 | 40 | 속도별 자기장 흡입. 고속에서는 off |
### C. 모듈 철학
- 기관차는 생존의 심장이다.
- 전투칸은 밖을 향한 이빨이다.
- 유지보수칸은 안쪽에서 버티게 하는 손이다.
- 채집칸은 느리게 달릴 이유를 만든다.
이 네 칸은 단순한 HP 덩어리가 아니라, 플레이어가 어떤 위험을 감수하고 어떤 리듬으로 달릴지 결정하게 만드는 감정 장치입니다.

## 5. 자원 시스템
### 자동 자원 채집
- 채집 모듈 주변 흡입 반경 내 자원이 자동으로 기차로 날아와 저장됩니다.
- 플레이어는 채집 조작 대신 **속도를 늦추는 전략적 판단**에 집중합니다.
- 채집 흡입 반경은 속도 0/1/2/3에 따라 `220 / 160 / 100 / 0 px`입니다.
- 고속 주행은 전장을 빠르게 통과시키지만, 채집의 가능성을 닫습니다.
### 자원 종류
| 자원 | MVP 용도 |
|------|----------|
| 고철 (Scrap) | 모듈 외벽 수리. 고철 10 = 장갑 +20 |
| 연료 (Fuel) | 이동 필수 자원. 시작 100, 0이면 게임 오버 |
| 냉각수 (Coolant) | 열 -40% 즉시 냉각. 시작 2개, 최대 4개 |
자원은 보상인 동시에 유혹입니다.
속도를 줄이면 더 많은 것을 얻지만, 그만큼 황무지도 더 오래 플레이어를 물어뜯습니다.

## 6. 이중 방어 시스템 (Dual Defense)
### 외부 방어
- 기차 밖 약탈자 차량, 사막 생물, 미니보스를 포탑으로 격퇴합니다.
- 외부 포탑은 자동으로도 작동하지만, 수동 조준이 더 강합니다.
- 마우스 클릭 지속 시 수동 사격: DPS 100% + 크리 25%
- 손을 떼면 자동 포탑: DPS 60%, 크리 없음
- 내부 모드 동안 외부 자동 포탑 효율은 60%로 유지되어, 밖을 완전히 버리지는 않지만 압박은 커집니다.
MVP 적 3종은 다음과 같습니다.
| 종 | HP | 행동 | 출현 페이즈 | 보상 |
|----|----|------|-------------|------|
| 약탈자 바이크 | 30 | 측면 추격, 단발 사격 (DPS 4) | Warm-up부터 | 고철 5 |
| 사막 벌레 | 80 | 지하 잠복 후 점프 충돌 (충격 25) | Crescendo부터 | 고철 12, 냉각수 1 |
| 코뿔 트럭 | 200 | 정면 돌진 (충격 60), 1회 미니보스 | Climax 3:30 | 연료 30, 고철 25 |
스폰 레이트는 `clamp(0.5 + timeMin * 0.6, 0.3, 3.5)` 마리/초를 기준으로 합니다.
### 내부 대응
외벽 장갑이 0이 되면 해당 칸에 내부 침입이 시작되고, 그 칸의 기능은 정지합니다.
| 대응 수단 | MVP 동작 |
|-----------|----------|
| 내부 모드 | `Tab`으로 전환 후 마우스로 침입자 사격 |
| 격리벽 (Bulkhead) | `Q`로 침입 칸 봉쇄, 기능 30초 정지 후 자동 해제 |
| 수리·냉각 대응 | 냉각수와 유지보수 버프로 화재·열 압박 완화 |
v1의 칸 분리(Eject)는 Post-MVP 확장 후보로 남깁니다.
MVP의 방어 감정은 "밖에서 포탄이 오고, 안에서 문이 뚫리는" 두 압박을 동시에 보는 데 있습니다.

## 7. 월드 맵 & 진행 방식
### MVP 진행 방식
MVP는 월드 맵 전체가 아니라, 하나의 위험 노드를 5분 동안 돌파하는 구조입니다.
| 단계 | 시간 | 의미 |
|------|------|------|
| Warm-up | 0:00-0:45 | 조작과 바이크 위협을 익히는 도입 |
| Crescendo | 0:45-2:15 | 적 2종 추가, 첫 이벤트, 자원 풍족 구역 |
| Pinch | 2:15-3:30 | 환경 카드, 내부 침입 강제, 열 80% 압박 |
| Climax | 3:30-4:30 | 코뿔 트럭, 추가 이벤트, BURNOUT 해금 대기 |
| Sprint | 4:30-5:00 | 결승선 가시화, 연료 점멸, 마지막 결단 |
승리 조건은 5분 생존 또는 진행도 100% 도달입니다.
패배 조건은 기관차 HP 0 또는 연료 0입니다.
### Post-MVP 월드 맵 비전
| 루트 | 특징 |
|------|------|
| 지름길 | 거리 짧아 연료 절약, 적 밀도 높아 엔진 풀가동 필요 |
| 우회로 | 거리 길어 연료 소모 크지만 자원 풍부, 상대적으로 안전 |
장기 구조에서는 시작점에서 최종 목적지까지 여러 갈래 경로가 존재합니다.
플레이어는 기차 상태, 연료, 장갑, 냉각수, 모듈 손상도를 보고 다음 노드를 선택합니다.
정비 구간은 노드 끝의 정착지에서 열리며, 자원으로 장갑 수리, 모듈 업그레이드, 칸 순서 재배치를 수행합니다.

## 8. 핵심 재미 요소 (Core Fun)
v2의 핵심 재미는 두 축으로 정리합니다.
### 축 1. 입력 긴장감 — 메인 재미
spec §4는 이 게임의 메인 흥미를 **입력 긴장감**으로 정의합니다.
플레이어는 한 손으로 속도를 바꾸고, 마우스로 외부 적을 쏘며, 필요할 때 `Tab`으로 내부 침입 대응에 들어갑니다.
| 입력 | 감정 |
|------|------|
| 1 / 2 / 3 / 0 | 지금 달릴지, 버틸지, 멈출지 결정 |
| 마우스 사격 | 자동 포탑보다 강한 개입감 |
| Tab | 밖을 잠시 포기하고 안쪽 위기를 보는 결단 |
| Q | 침입 칸을 봉쇄하는 손실 감수 |
| C | 냉각수 1개를 써서 열을 -40% 낮추는 응급 처치 |
| Space | 4분 이후 BURNOUT, 단 한 번의 폭주 |
BURNOUT은 240초 이후 1회 해금됩니다.
효과는 5초간 속도 +1, 사격 속도 2배, 모든 적 명중률 -50%입니다.
대신 종료 후 무작위 모듈에 강제 화재가 발생합니다.
이 스킬의 의도는 단순한 필살기가 아니라, **"화끈한 한 방 + 후유증 관리"**라는 결단 압박입니다.
### 축 2. 랜덤 이벤트 — 보조 재미
spec §5는 보조 흥미를 **랜덤 이벤트 카드**로 정의합니다.
이벤트는 6장 풀에서 1런당 1~2장 발동합니다.
발동 타이밍은 `45..105초` 구간과 `180..220초` 구간입니다.
| ID | 이름 | 효과 |
|----|------|------|
| E1 | 모래폭풍 | 25초간 시야 50% 감소, 적 명중률 -30%, 채집 흡입 -50% |
| E2 | 협곡 좁힘 | 20초간 외부 포탑 사거리 -30%, 적은 하단 한쪽만 출현 |
| E3 | 고철 더미 | 12초간 흡입력 2배, 적 스폰 1.5배 |
| E4 | 유령 신호 | 가짜 적 표시, 진짜 적 1마리 잠복 출현 |
| E5 | 냉각수 누수 | 30초간 열 +50% 가속, 채집물 1.5배 |
| E6 | 무전 SOS | 일시정지 선택: 5초 정지·연료 -10·고철 +30 또는 무시 |
랜덤 이벤트는 게임의 주인공이 아닙니다.
주인공은 언제나 속도, 조준, 내부 대응 사이에서 손이 바빠지는 입력 긴장감입니다.
이벤트는 그 손의 우선순위를 흔드는 황무지의 날씨입니다.

하지만 이 게임의 차별점은 "기다리는 디펜스"가 아니라 "달리면서 난이도를 직접 밟는 디펜스"라는 점입니다.

## 10. MVP에서 검증된 것 (2026-04-27)
MVP는 5분 1노드 캡슐로 분리되어 구현·검증되었습니다.
검증된 범위는 다음과 같습니다.
| 항목 | 검증 내용 |
|------|-----------|
| 5분 1런 구조 | 300초 타이머, 진행도 100%, 클리어/실패 화면, 재시작 |
| 클리어/실패 조건 | 기관차 생존 시 클리어, 기관차 HP 0 또는 연료 0 시 실패 |
| 1+3 모듈 | 기관차, 전투칸, 유지보수칸, 채집칸의 역할과 HP·장갑 |
| 속도 4단 | 정지/저속/중속/고속의 연료·열·채집 반경 차이 |
| 적 3종 | 약탈자 바이크, 사막 벌레, 코뿔 트럭 |
| 이벤트 6장 | 모래폭풍, 협곡 좁힘, 고철 더미, 유령 신호, 냉각수 누수, 무전 SOS |
| BURNOUT | 4분 이후 1회 폭주, 5초 강화, 종료 후 화재 |
| 침입 | 외벽 0 또는 페이즈 압박에 따른 내부 침입 대응 |
| 내부/외부 모드 | Tab 전환, 외부 포탑 효율 저하, 내부 침입자 사격 |
| 격리벽 | Q 입력으로 침입 칸 봉쇄, 30초 기능 정지 |
이 검증의 의미는 단순히 "돌아간다"가 아닙니다.
5분 안에 플레이어가 실제로 속도, 열, 연료, 적, 침입, 이벤트를 동시에 보게 된다는 점이 확인되었습니다.
즉 v1의 감정 문장인 "흐름을 지배하는 자가 황무지를 돌파한다"는 MVP에서 조작 가능한 형태로 내려왔습니다.

## 11. 다음 단계 (Post-MVP)
MVP 이후 확장은 다음 순서를 우선합니다.
| 단계 | 목표 |
|------|------|
| 정비소 | 노드 종료 후 장갑 수리, 모듈 업그레이드, 재배치 판단 추가 |
| 다중 노드 | 지름길·우회로 선택, 경로별 위험·보상 차별화 |
| 메타진행 | 영구 업그레이드, 신규 모듈 해금, 장기 생존 동기 부여 |
| 스프라이트 | 도형+컬러 MVP를 실제 기차·적·황무지 비주얼로 치환 |
| BGM | 질주, 위기, 정비, 클라이맥스 감정선을 음악으로 보강 |
추가 후보:
- 칸 분리(Eject) 시스템 정식화
- 정비소 이벤트와 상인
- 노드별 환경 기믹
- 적 세력별 패턴 차별화
- 모듈 손상에 따른 UI·사운드 피드백 강화
Post-MVP의 핵심 원칙은 하나입니다.
새 기능은 반드시 절대 규칙을 더 선명하게 만들어야 합니다.
빠르게 달릴수록 잃는 것이 있어야 하고, 느리게 달릴수록 덮쳐오는 것이 있어야 합니다.
