# 미식가의 단호한 식탁 (The Gourmet's Decisive Table)

**부제:** 완벽하지 않으면 나갈 수 없다

---

## 1. 기획 배경 및 의도

일반적인 타이쿤(Tycoon) 게임의 공식인 **"어떻게 하면 더 많이 팔까?"** 를 정면으로 뒤집는 기획입니다.

플레이어는 경영자이자 **절대적인 심판자인 '미식가 주인'** 입니다. 돈을 많이 버는 것이 목적이 아니라, 자신의 까다로운 기준을 지키면서도 식당을 유지하는 **'완벽주의 경영'** 이 이 게임의 핵심입니다.

---

## 2. 개요 (Vision)

| 항목 | 내용 |
|------|------|
| 장르 | 타이쿤 / 경영 시뮬레이션 |
| 핵심 컨셉 | 자신의 미식 기준을 지키며 식당을 운영하는 완벽주의 경영 타이쿤 |
| 주인공 | 지독한 미식가이자 식당 주인 |

---

## 3. 절대 규칙 (The Grand Law)

> **"내 혀를 통과하지 못한 요리는 쓰레기다."**

모든 요리는 주방에서 완성된 직후 **손님에게 바로 가지 못하고**, 화면 중앙의 **검수대(Tasting Zone)** 에 멈춥니다.

플레이어(미식가)가 결정하기 전까지 절대 서빙될 수 없습니다.

### 두 가지 선택지

| 선택 | 결과 |
|------|------|
| ✅ 승인 (Pass) | 손님에게 서빙 → 수익 발생 |
| 🗑️ 폐기 (Trash) | 식재료비 손실, 수익 0원 → 명성 유지 |

---

## 4. 핵심 시스템

### 4-1. 요리 품질 등급 (Quality Grade)
주방장 숙련도 + 식재료 신선도에 따라 요리는 **1~100점** 으로 평가됩니다.

| 점수 | 등급 | 효과 |
|------|------|------|
| 90점 이상 | 명작 | 명성 폭발적 상승, VIP 손님 유입, 거액 팁 |
| 70~89점 | 타협점 | 수익 발생, 명성 유지 또는 미세 하락 |
| 70점 미만 | 실패작 | 서빙 즉시 명성 급락, 단골 이탈 |

### 4-2. 명성(Reputation) vs 현금(Cash)

이 게임의 핵심 딜레마입니다.

```
현금  → 월세·인건비·식재료비 지불 (승인해야 벌 수 있음)
명성  → 식당 등급·VIP 손님 유치 (폐기해야 지킬 수 있음)
```

**"현금이 부족할 때 75점짜리 요리를 눈 감고 내보낼 것인가,
명성을 지키기 위해 적자를 감수하며 요리를 쓰레기통에 처박을 것인가?"**

### 4-3. 손님의 인내심 (Patience Meter)
- 각 손님은 기다림 게이지를 가지고 있음
- 플레이어가 고민·폐기·재조리를 반복하는 동안 게이지가 감소
- 게이지 소진 → 손님 분노 퇴장 → 명성 타격

### 4-4. 미각 피로도 (Palate Fatigue)
- 요리를 연속해서 검수하면 피로도 누적
- 피로도 높을 때 → 요리 실제 점수가 가려지거나 잘못된 정보 표시
- **해결법:** 와인·탄산수·셔벗 등 '입가심' 아이템 소모 또는 잠시 검수 중단

---

## 5. 주방 조절 (Kitchen Control)

플레이어는 주방장에게 실시간으로 지시를 내릴 수 있습니다.

| 지시 | 효과 |
|------|------|
| "천천히, 정교하게" | 조리 시간 ↑, 고득점 확률 ↑ |
| "빨리, 대충" | 조리 시간 ↓, 저득점 확률 ↑ |

---

## 6. 미식가의 특수 능력 (Gourmet Skills)

| 스킬 | 설명 |
|------|------|
| 입가심 (Palate Cleanser) | 미각 피로도 초기화, 요리 점수 정확도 상승 |
| 독설 (Sharp Tongue) | 폐기 시 주방장에게 독설 → 다음 조리 고득점 확률 일시 상승 (단, 주방장 스트레스 ↑) |
| 재료 직관 | 조리 전 식재료 신선도 파악, 실패 가능성 높은 요리 선제 차단 |

---

## 7. 게임 진행 방식 (Core Loop)

```
1. 영업 준비
   └─ 주방장 배치 / 식재료 구매 / 메뉴 설정

2. 영업 개시 (실시간)
   └─ 주방장 자동 조리 → 속도 지시 내리기

3. 검수의 늪 (Peak)
   └─ 쏟아지는 접시들 → 순간 판단 연속
       "버려! 다시! 이건 통과! 버려!"

4. 영업 종료
   └─ 수익·명성 정산 → 주방 강화 → 1번 반복
```

---

## 8. 핵심 재미 요소 (Core Fun)

1. **권력의 맛** — 요리를 폐기할 때의 타격감 (접시 깨지는 소리, 처박히는 연출)
2. **아슬아슬한 경영** — 파산 직전 통장을 보며 예술적 자존심을 어디까지 굽힐지 결정
3. **성장의 기쁨** — 혹독한 검수를 견뎌낸 주방장이 점점 완벽에 가까운 요리를 내놓을 때의 뿌듯함
4. **완벽한 한 접시의 카타르시스** — 수많은 폐기 끝에 나온 100점짜리 요리가 서빙될 때의 극적 연출

---

## 9. 개발 포인트

이 게임은 **'폐기' 연출에 목숨을 걸어야 합니다.**

- 쓰레기통에 요리가 담길 때의 물리 엔진
- 주방장의 좌절 애니메이션
- 명성이 오를수록 식당 인테리어가 점점 고결하고 화려하게 변화하는 시각적 피드백

이 세 가지가 이 게임의 완성도를 결정합니다.
